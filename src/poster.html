Day 1 - Fundamentals: Deep Learning, Learning Theory, Convolutions, Embeddings.

The first day provides a solid base of knowledge, by covering key concepts through group learning and exercises.

Attendees will implement a reference model in TensorFlow, and learn the necessary components of all deep learning training pipelines.

Day 2 - Hands-On TensorFlow: TensorBoard, Training, Evaluation.

On the second day we will work through the details of training models in TensorFlow in more detail. We will build a reference model, and learn how to train our model, check it's progress, and evaluate it's accuracy.

Day 3 - Group Projects.

The last day provides hands-on experience in deep learning model development, in TensorFlow.

Participants will consolidate their knowledge by working in small groups on independent projects. Project options vary across image processing, embeddings, and generative models.

The workshop in session!
Questions you'll be able to answer after taking this workshop:
  • How does deep learning differ from machine learning and normal programming?
  • How do you build deep learning models in TensorFlow?
  • What are some common deep learning architectures, and how can I apply them?
  • How do I build a good data set for training? How much data do I need?
  • How do I process text with deep learning? What about images? What about both, jointly?!
